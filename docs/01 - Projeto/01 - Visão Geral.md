# Visão Geral

#senai #chamada #overview

> [[00 - Índice|← Índice]]

Sistema web para controle de chamada (presença) de alunos do **SENAI Limeira**.

---

## O que faz

- Professores fazem login com a conta Microsoft institucional
- Selecionam turma e disciplina
- Marcam presença — por padrão **todos presentes**, professor desmarca os ausentes
- Dados salvos nas listas do SharePoint existentes
- Coordenadores podem importar alunos via Excel
- Relatórios de presença por turma e data

## Contexto

Projeto real para uso interno no SENAI.
Reaproveitamento da infraestrutura **Microsoft 365** já contratada — sem custo adicional de banco de dados ou autenticação.
